## 프로젝트 설치

```
yarn install
```

### 프로젝트 컴파일 및 핫 리로드

```
yarn serve
```

### 프로젝트 빌드

```
yarn build
```

### ESLint 및 파일 수정

```
yarn lint
```

## 사용된 라이브러리

- [Vue3](https://github.com/vuejs/vue-next)
- [Graphql](https://graphql.org/)
- [Apollo](https://www.apollographql.com/)
- [Vue-Apollo](https://v4.apollo.vuejs.org/)
